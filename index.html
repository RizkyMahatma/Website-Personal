front end

Front-end development adalah proses membuat antarmuka pengguna website agar interaktif, responsif, dan menarik menggunakan HTML, CSS, dan JavaScript.

Back End

Back-end development adalah proses membangun logika, database, dan server agar aplikasi web dapat berjalan, menyimpan, dan memproses data dengan aman.

DevOps

DevOps adalah praktik kolaborasi antara developer dan IT ops untuk otomatisasi, integrasi, dan pengiriman aplikasi yang cepat, stabil, dan efisien.

Debugger

Debugger adalah alat untuk mendeteksi, menganalisis, dan memperbaiki kesalahan (bug) dalam kode program secara sistematis selama proses pengembangan.
